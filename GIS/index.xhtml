Scene initialization values needs to be set first. Layer selection is mandatory. Additionally number of terrain grids and used texture and texture resolution can be changed. After selections are done, terrain can be loaded by pressing "Load selected layer data"-button. Before changing initialization values "Reset view"-button needs to be used first.
